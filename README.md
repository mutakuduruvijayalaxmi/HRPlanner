# HR Planner

HR Planner is a simple web-based Human Resource management application designed to help organizations manage employees, candidates, attendance, tasks, and job postings from a centralized dashboard.

## Overview

This project demonstrates a structured HR management interface built using HTML, CSS, and JavaScript. It provides multiple modules that simulate common HR operations such as tracking employees, managing candidates, recording attendance, and generating reports.

## Features

* Dashboard with quick overview of HR activities
* Employee management interface
* Candidate tracking system
* Attendance monitoring
* Task management module
* Job postings management
* Notification system
* Reports section for HR insights
* Light/Dark theme toggle
